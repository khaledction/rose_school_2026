# 01) متطلبات جهاز البناء (الذي يحوّل المشروع إلى تنفيذي)

هذا الجهاز هو الذي فيه المشروع + Flutter، ومنه نُخرج `RoseSchoolSetup.exe`.

## أ) نظام التشغيل
- Windows 10 أو Windows 11
- **64-bit (x64)** فقط

## ب) البرامج الأساسية

### 1) Git
- للسحب/الدفع من GitHub
- https://git-scm.com/download/win

تحقق:
```powershell
git --version
```

### 2) Flutter SDK
- https://docs.flutter.dev/get-started/install/windows
- أضِف Flutter إلى PATH

تحقق:
```powershell
flutter --version
flutter doctor
```

### 3) Visual Studio (مهم جدًا لبناء Windows)
ثبّت **Visual Studio 2022** (Community كافية) مع workload:

- **Desktop development with C++**

مكونات مفيدة داخلها:
- MSVC v143
- Windows 10/11 SDK
- C++ CMake tools

تحقق:
```powershell
flutter doctor
```
يجب أن يظهر Windows toolchain بدون أخطاء حرجة.

### 4) تفعيل Windows Desktop في Flutter
```powershell
flutter config --enable-windows-desktop
flutter devices
```
يجب أن ترى `windows`.

### 5) Inno Setup 6 (للـ Installer الأنيق)
- https://jrsoftware.org/isinfo.php
- بعد التثبيت يجب وجود:
  - `C:\Program Files (x86)\Inno Setup 6\ISCC.exe`

### 6) اتصال إنترنت (أثناء البناء)
مطلوب مرة واحدة على الأقل لتحميل:
- حزم `pub`
- `vc_redist.x64.exe` (إن لم يكن موجودًا)

## ج) مساحة القرص المقترحة
- 10 GB+ فارغة على الأقل
- SSD أفضل للبناء الأسرع

## د) صلاحيات
- حساب يقدر يشغّل PowerShell
- صلاحية Administrator عند بناء/تثبيت المثبت

## هـ) ما لا تحتاجه على جهاز البناء؟
- لا تحتاج Android Studio لبناء ويندوز
- لا تحتاج Xcode
- لا تحتاج خادم/استضافة

## و) تحقق سريع قبل أول بناء
```powershell
flutter doctor -v
where flutter
where git
Test-Path "${env:ProgramFiles(x86)}\Inno Setup 6\ISCC.exe"
```
