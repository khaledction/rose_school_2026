# 03) متطلبات الجهاز الهدف (أي لابتوب/ديسكتوب سيشغّل البرنامج)

الجهاز الهدف **لا يحتاج Flutter** ولا Visual Studio.

## الحد الأدنى
- Windows 10 أو Windows 11
- معمارية **64-bit (x64)**
- 4 GB RAM (8 GB أفضل)
- 300–500 MB مساحة فارغة على الأقل
- شاشة تدعم استخدام Desktop عادي

## ما يثبّته/يشمله التوزيع

### إذا استخدم `RoseSchoolSetup.exe`
- يثبّت التطبيق
- يثبّت **VC++ Redistributable x64 تلقائيًا** عند الحاجة
- ينشئ اختصارات

### إذا استخدم ZIP المحمول
- يجب الإبقاء على كل ملفات المجلد معًا
- داخل ZIP يوجد:
  - `rose_school.exe`
  - DLLs + `data`
  - `vc_redist.x64.exe` (للتشغيل اليدوي إن لزم)
  - `README_PORTABLE.txt`

## ما لا يحتاجه الجهاز الهدف
- لا Git
- لا Flutter
- لا Android Studio
- لا إنترنت (بعد اكتمال التثبيت/نسخ الملفات)

## توصية عملية
- للإدارة/المدرسة: استخدم **Setup**
- للنسخ السريعة الداخلية: ZIP مقبول
