# 05) قائمة تحقق قبل إرسال النسخة لأي جهاز

## على جهاز البناء
- [ ] `flutter doctor` سليم لمحور Windows
- [ ] Inno Setup 6 مثبت
- [ ] تم السحب لآخر `main`
- [ ] نجح:
  ```powershell
  powershell -ExecutionPolicy Bypass -File .\scripts\build_release_installer.ps1
  ```
- [ ] موجود `dist\RoseSchoolSetup.exe`
- [ ] موجود ZIP في `dist\`
- [ ] `installer\redist\vc_redist.x64.exe` موجود

## تجربة سريعة قبل التوزيع
- [ ] تثبيت Setup على جهازك
- [ ] فتح التطبيق بنجاح
- [ ] تسجيل دخول
- [ ] فتح الأقساط/المستحقين
- [ ] إضافة قسط/دفعة
- [ ] التأكد أن المستحق يتحول أخضر

## على الجهاز الهدف
- [ ] Windows 10/11 x64
- [ ] تشغيل `RoseSchoolSetup.exe` كمسؤول إن لزم
- [ ] إذا ظهرت DLL: انتظر التثبيت التلقائي أو ثبّت `vc_redist.x64.exe`
- [ ] إعادة تشغيل الجهاز عند الحاجة
- [ ] فتح اختصار **مدرسة روز**

## لا ترسل
- [ ] لا ترسل `exe` وحده من مجلد Release
- [ ] لا ترسل مجلد المشروع كاملًا للمستخدم النهائي (إلا للمطورين)
