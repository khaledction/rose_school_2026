# 02) خطوات البناء والتثبيت (من الكود إلى Setup.exe)

## 1) مزامنة المشروع
```powershell
cd C:\Users\khaledction\Desktop\new-rose
git fetch origin
git reset --hard origin/main
git clean -fd
```

## 2) البناء الكامل بأمر واحد (موصى به)
```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build_release_installer.ps1
```

السكربت يقوم بـ:
1. تنزيل `vc_redist.x64.exe` تلقائيًا إلى `installer\redist\`
2. `flutter clean` (يمكن تخطيه بـ `-SkipClean`)
3. `flutter pub get`
4. `flutter build windows --release`
5. إنشاء ZIP محمول في `dist\`
6. بناء `dist\RoseSchoolSetup.exe` عبر Inno Setup

### خيارات
```powershell
# بدون clean
powershell -ExecutionPolicy Bypass -File .\scripts\build_release_installer.ps1 -SkipClean

# ZIP فقط بدون installer
powershell -ExecutionPolicy Bypass -File .\scripts\build_release_installer.ps1 -SkipInstaller
```

## 3) أين المخرجات؟
| الملف/المجلد | الاستخدام |
|--------------|-----------|
| `dist\RoseSchoolSetup.exe` | تثبيت أنيق (يثبّت VC++ تلقائيًا عند الحاجة) |
| `dist\RoseSchool2026_Portable_....zip` | تشغيل محمول |
| `build\windows\x64\runner\Release\` | مجلد التنفيذ الخام |
| `installer\redist\vc_redist.x64.exe` | مكتبة VC++ المستخدمة داخل المثبت |

## 4) طريقة يدوية (بدون السكربت)
```powershell
cd C:\Users\khaledction\Desktop\new-rose
flutter clean
flutter pub get
flutter build windows --release
```
ثم:
1. افتح `installer\RoseSchool.iss` بـ Inno Setup
2. تأكد وجود `installer\redist\vc_redist.x64.exe`
3. Compile

## 5) التوزيع
### الأفضل
أرسل فقط:
```text
RoseSchoolSetup.exe
```

### البديل
أرسل ZIP المحمول، ويُفك كاملًا ثم:
```text
rose_school.exe
```

## 6) ماذا يفعل المثبت الآن؟
- يثبّت التطبيق
- ينشئ اختصار عربي: **مدرسة روز 2026**
- يستخدم أيقونة مخصصة
- يعرض ترحيب + ترخيص + إنهاء
- **يثبّت VC++ Redistributable x64 تلقائيًا إذا كان ناقصًا**
