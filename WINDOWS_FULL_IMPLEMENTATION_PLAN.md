### غير المكتمل أو غير المربوط
1. **التقارير/التصدير الحقيقي من الواجهة**
   - `ExportService` موجود (JSON/CSV) لكن **لا أزرار** في مركز البيانات تستدعيه.
   - حزمة `excel` في pubspec لكن المسار الحالي CSV.
2. **النسخ الاحتياطي**
   - Backup/Restore موجودان في `BackupService`.
   - صيغة الملف `.zip` اسمياً لكنها **JSON+base64** (لا حزمة `archive`).
3. **حماية المدير**
   - `AdminPasswordDialog` مكتوب وغير مربوط بـ backup/restore/export.
4. **Dark Mode**
   - Provider موجود؛ غير مدمج في `main.dart` ولا زر فعّال في الهيدر.
5. **فتح المجلدات على ويندوز**
   - حالياً ينسخ المسار / snack فقط؛ المطلوب `explorer`.
6. **الأرشفة التلقائية**
   - غير منفّذة.
7. **اختبارات Windows workflow**
   - Checklist جاهز (83 حالة) — لم يُملأ بعد تشغيل محلي.

### المرحلة C — النسخ الاحتياطي والاستعادة 🟡
**صيغة مستهدفة:** `ROSE_BACKUP_YYYY_MM_DD_HHMMSS.zip`  
**المحتويات المطلوبة:** database + snapshot JSON + files/ + manifest + preview  
**الوضع:** البنية المنطقية موجودة؛ التعبئة ليست ZIP ثنائياً حقيقياً بعد.

4. `data_snapshot.json`
5. ملخص مالي

**ملاحظة:** Excel للتقارير؛ JSON للنسخ؛ SQLite المصدر الأساسي.
